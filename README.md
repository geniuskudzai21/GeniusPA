# Genius Performance Analysis

Genius Performance Analysis is a web application designed to help students track, analyze, and visualize their academic performance. The app allows users to input academic details, view statistics, set goals, and celebrate achievements, all in a modern and responsive interface.

Key Features
- Input and manage academic modules with marks and classifications
- Dynamic statistics and visualizations (bar and pie charts)
- Achievement tracking with medal awards
- Goal setting and progress tracking
- Download academic details as CSV
- Responsive sidebar navigation

Technologies Used
- HTML, CSS (responsive design)
- JavaScript (localStorage, Chart.js, jsPDF)

Purpose
The application empowers students to understand their academic progress, set goals, and receive insights for future improvement and career recommendations.
